## 📖 About the Project
**NyayaVeda AI** is a Legal-Tech platform. Our goal is to make legal information easy to understand for everyone in India. We use AI to read, analyze, and simplify legal documents and rights.

This repository contains the main code and API for the NyayaVeda platform.

---

## ✨ Key Features
* **🧠 Smart Legal AI:** AI models specially trained for Indian laws.
* **⚡ Fast API:** Built with FastAPI to handle many users quickly.
* **🔒 High Security:** Strict data privacy for all user queries.
* **🌍 Fast & Scalable:** Hosted on Vercel and Cloudflare for zero downtime.

## 🚀 How to Run Locally

### What you need:
* Python 3.10+
* Node.js 18+
* Git

### Steps to Install:

1. **Clone the code:**
    ```bash
    git clone [https://github.com/NyayaVeda-Ai/nyayaveda-core.git](https://github.com/NyayaVeda-Ai/nyayaveda-core.git)
    cd nyayaveda-core
    ```

2. **Setup Backend:**
    ```bash
    cd backend
    python -m venv venv
    source venv/bin/activate
    pip install -r requirements.txt
    ```

3. **Run the Server:**
    ```bash
    uvicorn main:app --reload
    ```
